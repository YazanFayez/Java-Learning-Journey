# Calculator Project
This is a simple Java-based calculator that performs basic arithmetic operations.

## Files
* `Calculator.java`: Contains the logic for addition, subtraction, multiplication, and division.
* `TestClac.java`: The driver class used to run and test the calculator.

## How to Run
1. Open both files in jGRASP.
2. Compile `Calculator.java` first.
3. Run `TestClac.java`.
